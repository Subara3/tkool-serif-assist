# RPGツクールMZ イベントコマンド パラメータリファレンス

公式PDF (https://rpgmakerofficial.com/product/mz/plugin/javascript/script_reference/eventcode.pdf) より抽出。

## メッセージ系コマンド

### code 101: 文章の表示
```
parameters: [顔ファイル名, 顔インデックス, 背景, ポジション, 名前]
```
- 0: 顔グラフィックファイル名 (string) — `""` で顔なし
- 1: 顔グラフィックインデックス (int 0-7)
- 2: 背景 (int)
  - 0: ウィンドウ
  - 1: 暗くする
  - 2: 透明
- 3: ポジション (int)
  - 0: 上
  - 1: 中
  - 2: 下
- 4: 名前 (string) — `""` で名前なし

### code 401: 文章の表示（メッセージ内容）
```
parameters: [メッセージ文字列]
```
- 1行分のテキスト。101の後に複数個並ぶ。

### code 102: 選択肢の表示
```
parameters: [選択肢配列, キャンセルタイプ, デフォルトタイプ, 背景, ポジション]
```
- 0: 選択肢配列 (array of string)
- 1: キャンセルタイプ (-2:分岐, -1:禁止, 0..:各選択肢)
- 2: デフォルトタイプ (-1:なし, 0..:各選択肢)
- 3: 背景 (0:ウィンドウ, 1:暗くする, 2:透明)
- 4: ポジション (0:左, 1:中, 2:右)

### code 402: 選択肢の表示（**のとき）
```
parameters: [選択肢番号]
```

### code 403: 選択肢の表示（キャンセルのとき）
パラメータなし

### code 103: 数値入力の処理
```
parameters: [変数, 桁数]
```

### code 104: アイテム選択の処理
```
parameters: [変数, アイテムタイプ]
```
- アイテムタイプ: 0:通常アイテム, 1:大事なもの, 2:隠しアイテムA, 3:隠しアイテムB

### code 105: 文章のスクロール表示
```
parameters: [速度, 早送りなし]
```

### code 405: 文章のスクロール表示（メッセージ内容）
```
parameters: [メッセージ文字列]
```

## 終端

### code 0: コマンドリスト終端
```
parameters: []
```

## 顔グラフィック インデックス対応

顔シートは 4列×2行 の8分割:
```
0  1  2  3
4  5  6  7
```

## MZデフォルト顔ファイル名

- Actor1 — 主要キャラ8人
- Actor2 — 主要キャラ8人
- Actor3 — サブキャラ8人
- Actor1_0 〜 Actor1_7 — Actor1の個別ファイル
- Actor2_0 〜 Actor2_7 — Actor2の個別ファイル
- etc.
